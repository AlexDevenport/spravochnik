Необходимые инструменты для запуска:
- Docker;
- Утилита make (по возможности).

Первым делом необходимо будет запустить докер-контейнер командой "make build-docker" для сборки проекта. Таким образом создастся образ проекта, подгрузится и заполнится тестовыми данными БД.

Теперь можно перейти в документацию в виде swagger по адресу:
http://127.0.0.1:8000/docs#

Здесь есть эндпоинты для создания (POST) и получения (GET) инцидентов.

Для взаимодействия с ручками необходимо вводить секретный ключ "supersecret". Данный ключ находится в файле ".env".

Все остальные команды расписаны в файле Makefile.
